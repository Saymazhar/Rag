# RAG YouTube QA System

This project implements a Retrieval-Augmented Generation (RAG) pipeline using:

- LangChain
- FAISS
- HuggingFace Embeddings
- Groq LLM

## Features
- Extract transcript from YouTube
- Chunking and embedding
- Vector search with FAISS
- Question answering
